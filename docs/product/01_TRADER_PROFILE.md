# Trader Profile

The Trader Profile is the center of the trader experience.

## Owner view

Visible to the trader:
- Edit profile
- Upload photos
- Manage strategies
- Publish research
- Enter challenge
- View revenue
- Settings

## Allocator view

Visible to investors:
- Profile
- Photos
- Strategy stats
- Performance
- Research
- Challenge history
- Copy button

## Principle

Same profile. Two modes.

- Owner mode = manage reputation.
- Allocator mode = decide whether to allocate capital.
