# Screen Responsibilities

## BullPad

Question:
How is my money doing?

Contains:
- Balance
- Portfolio
- Performance
- Deposit
- Withdraw
- Top Traders
- Chat
- Research feed
- Copy status

Does not contain:
- Trader edit tools
- Strategy creation
- Challenge management

## Trader Dashboard

Question:
How do I grow my trader business?

Contains:
- Challenge
- Strategies
- Research
- Revenue
- Settings
- Performance admin

Does not contain:
- Allocator portfolio
- Deposit/withdraw as investor

## Trader Profile

Question:
Why should someone trust this trader?

Contains:
- Hero
- Bio
- Trust metrics
- Strategies shelf
- Research preview
- Gallery
- Activity timeline

Does not contain:
- Copy button directly
- Private revenue
- Private settings

## Strategy Profile

Question:
Should I copy this strategy?

Contains:
- ROI
- Equity curve
- Drawdown
- Win rate
- Challenge rank
- Risk
- Research
- Copy CTA
