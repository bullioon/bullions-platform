# Bullions Product Map

Bullions has two primary experiences.

## 1. BullPad — Allocator

Purpose: manage money.

Includes:
- Balance
- Portfolio
- Performance
- Deposit
- Withdraw
- Top 5
- Trader chat
- Research feed
- Copy status

## 2. Trader Profile — Trader

Purpose: build reputation and attract capital.

Includes:
- Public profile
- Photos / gallery
- Strategies
- Performance
- Research
- Challenge
- Revenue
- Settings

## Main bridge

BullPad → Top 5 → Trader Profile → Copy
