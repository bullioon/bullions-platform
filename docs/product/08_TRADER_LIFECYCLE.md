# Trader Lifecycle

## 1. Become Trader

User creates trader identity:
- Display name
- Username
- Avatar
- Bio
- Markets
- Trading style

## 2. Buy Challenge

Trader chooses one challenge license:

### 50K Demo
Entry fee: $350

### 200K Demo
Entry fee: $1,080

Payment:
- Phantom

## 3. Receive Demo Account

Challenge account state:
- waiting_payment
- payment_confirmed
- waiting_account
- trading
- completed
- disqualified

## 4. Launch First Strategy

Trader starts with one public strategy slot.

Future unlocks:
- Slot 2
- Slot 3

Max active public strategies:
- 3

## 5. Compete

Strategy receives:
- Performance snapshots
- Challenge score
- Rule checks
- Rank

## 6. Top 5

Top 5 strategies become allocator-facing.

Revenue:
- Rank 1: 35%
- Rank 2-5: 25%

## 7. Revenue

Trader earns from Sunday allocator withdrawals.
